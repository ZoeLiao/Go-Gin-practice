# Go-Gin-practice
Gin + swag (Swagger) + gorm (CRUD) + testify + log

## Set up
- `go run main.go`
- Visit [http://localhost:8080/swagger/index.html](http://localhost:8080/swagger/index.html)

## Swagger
- Swagger: `swag init`

## Check
- `brew install pre-commit`
- `pre-commit install`
- `pre-commit run --all-files`

## Test
- `go test tests/*.go`
